INTRODUCCION

Este proyecto esta basado en crear un robot con la funcion de mover objetos que obstruyen el paso, realizaremos las siguientes actividades para llegar a ello: programacion de pagina web y del robot utilizando materiales como, sensores de evitacion de obstaculos (KY-032)y sensor de impacto (KY-031), la contruccion del robot mediante piezas metalicas y plasticas para lo que seria su cuerpo y funciones.

MODO DE USO En la pagina colocaremos contenido sobre el robot como fotos y videos. Mostraremos el proceso de realizacion de este y su funcionamiento. Pondremos informacion sobre lo que se utilizo para su desarrollo y futuras versiones y/o actualizaciones sobre el modelo del robot para los futuros clientes que esten interesados en el mismo y sus usos.

ROBOT

Para codificar el robot utilizaremos sensores y tendremos un sistema de monitoreo para observar el avance del robot y sus posibles fallas. Este movera objetos lijeros y los llevara a un lugar de descarga donde ya no esten obstruyendo el paso, lo programaremos para que se mueva en diferentes direcciones como de derecha, izquierda, adelante, atras y para que gire sobre su propio eje.
